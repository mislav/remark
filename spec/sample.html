Remark — HTML to Markdown converter

This is a sample document which will get updated as Remark understands more HTML. It reflects what's currently supported.

Known block elements are left intact if they have attributes. Markdown doesn't have a syntax for them.

Elements that can't be represented in Markdown are left intact.

SCRIPT and HEAD tags are swallowed, as browsers don't render them as content.
